# Mentors without borders - Admin

## Getting Started on development

### One time setup

#### for Mac users

```bash
brew install nvm
nvm use
```

### Run the development server

```bash
npm install
npm run dev

```

### To run production build locally

```bash
npm install
npm run build
npm start
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
